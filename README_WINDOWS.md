**Запуск на Windows:**
1) Встанови Python 3.10+.
2) Відкрий PowerShell/CMD у цій папці.
3) python -m venv venv
4) venv\Scripts\activate
5) pip install -r requirements.txt
6) Встав токен у run_windows.bat (або run_windows_background.bat)
7) Запусти run_windows.bat
